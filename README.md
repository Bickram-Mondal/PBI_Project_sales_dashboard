# Sales Analytics Dashboard - Power BI

## Overview

This project demonstrates the development of an end-to-end Sales Analytics Dashboard in Power BI. The solution leverages a relational data model built using Customers, Orders, Order Details, Products, Categories, and a custom Freight by Ship Country table to generate actionable business insights.

## Features

* Data modeling with fact and dimension tables
* DAX measures and calculated columns for KPI generation
* Interactive slicers for Country, Month, Quarter, and Year
* Sales analysis by customer, product, and category
* Freight cost analysis by shipping country
* KPI tracking for Net Sales, Orders, Customers, and Products
* Dynamic and user-friendly dashboard design

## Data Model

The project follows a star-schema-inspired design where Orders and Order Details serve as transactional data sources, while Customers, Products, and Categories act as supporting dimensions. An aggregated freight table is used to analyze shipping-related costs across countries.

## Dashboard Insights

* Top-performing customers by sales
* Category-wise sales contribution
* Product-level sales distribution
* Freight trends across shipping countries
* Business KPI monitoring through interactive visualizations

## Tools & Technologies

* Power BI Desktop
* Power Query
* DAX
* Data Modeling
* Data Visualization

This project showcases the complete workflow from data modeling and transformation to business reporting and dashboard development.
